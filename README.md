# TI Store - E-commerce de Smartphones

A **TI Store** é uma plataforma de e-commerce simplificada, desenvolvida para a venda de smartphones de última geração. O projeto foca em uma interface limpa e intuitiva, permitindo que o usuário navegue entre os lançamentos mais recentes da Apple e acesse notícias sobre o mercado tecnológico.

##  Objetivo do Sistema

Este projeto foi criado com o intuito de aplicar conceitos fundamentais de desenvolvimento web front-end, simulando uma loja real. Os principais objetivos incluem:
* Apresentar um catálogo de produtos (iPhones) de forma organizada.
* Fornecer navegação entre páginas (Início, Produtos, Carrinho e Contato).
* Demonstrar uma seção de notícias integrada para manter o consumidor informado sobre avanços tecnológicos.
* Criar uma interface responsiva e amigável ao usuário.

##  Tecnologias Utilizadas

O sistema foi construído utilizando as seguintes tecnologias:

* **HTML5:** Estruturação semântica de todo o conteúdo das páginas.
* **CSS3:** Estilização visual, incluindo o uso de classes para botões e containers flexíveis para os produtos.
